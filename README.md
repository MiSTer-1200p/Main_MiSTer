# MiSTer 1920x1200 Fork - Aitor Gomez variant

Based on [spark2k06/Main_MiSTer](https://github.com/spark2k06/Main_MiSTer) with 1920x1200 framebuffer support.

See the [main branch](https://github.com/MiSTer-1200p/Main_MiSTer/tree/master) for full documentation about the 1200p changes.

## Installation

```ini
[distribution_mister]
db_url = https://raw.githubusercontent.com/MiSTer-1200p/Main_MiSTer/db/aitorgomez/db.json.zip
```
